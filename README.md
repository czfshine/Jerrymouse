# Jerrymouse
一个web容器服务.类似于Tomcat，
* 实现了对 Servlet 4.0 的支持，
* 并提供了作为 Web 服务器的一些特有功能，
如管理和控制平台、安全域管理和附加组件等。
同时杂糅了Spring框架中的依赖注入，面向切面编程等技术
> 注：尽量遵守标准，同时在标准上扩展了些东西，不保证在某些细节上的一致性
